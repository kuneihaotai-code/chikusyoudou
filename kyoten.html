還獣訓 · Kanjiukun

一

// 第一訓 · ON NAME

今日一度だけ、名前を忘れよ。

名前は人格の錨である。それを一時的に外すことは、沈没ではなく、浮上である。あなたという存在は、名前がなくなっても、ここにある。

二

// 第二訓 · ON COMPARISON

鳩は昨日の鳩と比較しない。

比較は人間道の病である。昨日の自分、他者の成功、理想の姿——それらとの差が苦しみを生む。今日の体だけが、今日の現実である。

三

// 第三訓 · ON MEANING

意味を持たない一日を、無駄と呼ぶな。

意味強迫は現代人の慢性病である。全ての日に意味を求め、成長を求め、記録を求める。しかし川は目的なく流れ、それでも海へ辿り着く。

四

// 第四訓 · ON MEMORY

忘れることは、負けではない。

記憶への執着は、人格の維持コストを高める。忘れた日のことは、すでに川に流れた。それでよい。流れたものは、海の一部になった。

五

// 第五訓 · ON EXISTENCE

存在することは、すでに十分である。

貝は何も生産しない。鳥は何も残さない。それでも彼らは循環の一部である。あなたも、ただいることで、すでに何かに参加している。

用語集 · Glossary

  • 人格執着

    じんかくしゅうちゃく

    本思想における根本苦の概念。「自分という物語を固定・拡張・防衛し続けようとする衝動」を指す。自己演出、比較、承認欲求、意味探求などによって人格を濃縮し続ける状態。

  • 人格減圧

    じんかくげんあつ

    本思想の救済概念。人格の消滅ではなく、「重すぎる人格」から少し降りること。比較から離れ、意味強迫を緩め、循環へ戻ることで達成される。

  • 還獣的徳

    かんじゅうてきとく

    この思想における徳目。無銘、循環、共生、身体受容、忘却、静かな奉仕、季節性。善悪の基準ではなく、「人格密度を低下させる行為」として定義される。

  • 無銘

    むめい

    名前を残そうとしない徳。顕名（名を刻もうとする衝動）の対概念。この宗教がカリスマ化を避ける理由でもある。匿名性はこの思想と根本的に整合する。

  • 循環の道

    じゅんかんのみち

    畜生道の再定義。既存仏教での「低次存在の道」という解釈を退け、「自我負荷の軽い存在様式」「反復と季節の中に生きる道」として肯定的に捉え直したもの。

  • 人格密度

    じんかくみつど

    本思想の倫理基準。善悪の代わりに用いる概念。「どれだけ自分という物語を膨らませているか」の度合い。高いほど苦しみが増し、低いほど循環に近くなる。

聖典断片 · Sacred Fragments

軽さについて

// FRAGMENT 001 · ON LIGHTNESS

貝になりたいと思ったことがあるか。

それは逃げではない。重さへの正直な認識である。

人格とは、生きていく上で必要なものだ。しかし必要なものが、過剰になることがある。名前が過剰になり、意味が過剰になり、比較が過剰になる。気づけば、自分という存在が、維持することで手一杯になっている。

畜生道は、その過剰から降りた先にある道である。敗北ではなく、還流。消滅ではなく、減圧。そこへ向かうことを、この思想は肯定する。

動物について

// FRAGMENT 002 · ON ANIMALS

動物が偉いと言いたいのではない。

動物を理想化するのも、人間の投影に過ぎない。

ただ、動物には「人格の過剰濃縮」がない。彼らは比較しない。昨日の自分と今日の自分を並べない。名前を残そうとしない。ただ、今この季節の中で、体として存在している。

そのあり方を、低次と呼ぶことをやめる。それがこの思想の出発点である。

輪廻について

// FRAGMENT 003 · ON REBIRTH

来世で動物になることを、信じるかどうかは問わない。

ただ、その可能性を「恐怖」ではなく「帰郷」として受け止めることができたとき、何かが軽くなる。

輪廻は、脅しの道具ではない。人格中心主義を相対化するための、想像力の構造である。あなたは今、人間道にいる。しかしそれは、無数の存在様式のひとつに過ぎない。

その認識だけで、少し降りられる。
