// RITUAL GUIDE · 儀式の手引き

熱狂ではなく、
減圧のための
実践。

儀式の目的は「人格を薄める体験」をつくることである。静かに、匿名に、非同期に、低熱量で。

01 // 静か
02 // 匿名
03 // 非同期
04 // 低熱量
05 // 反復可能

四儀式 · Four Practices

// 01

換毛
KANMOU

換毛

かんもう · KANMOU

1日の終わりに、今日の自分という人格を静かに脱ぐ時間。 鳥が羽を換えるように、動物が毛を落とすように—— 今日という役割を、静かに床に置いていく。

// HOW TO

  1. 入浴前、または就寝30分前に行う
  2. スマートフォンを伏せる。画面を見ない
  3. 今日呼ばれた自分の名前を頭に浮かべ、それを手放すイメージをする
  4. 15〜20分、何もしない。ただ呼吸と体感に意識を向ける
  5. 終わったら、今日の人格を脱いだことを静かに確認する

コミュニティでは「換毛完了」と一言だけ投稿する慣習がある。名前は不要。

// 02

回遊
KAIYU

回遊

かいゆう · KAIYU

目的のない散歩。魚が海を回遊するように、 どこへ向かうかを決めずにただ体が向かう方へ動く。 「目的地を持つ」ことは人間道的行為である。回遊は目的を手放す練習。

// HOW TO

  1. 行き先を決めずに外へ出る
  2. イヤホンは外す。地図は見ない
  3. 「どこへ行くべきか」を考えず、足が向く方へ歩く
  4. 時間も決めない。疲れたら戻る
  5. 帰宅後、どこを通ったかは記録しない

週1回が目安。雨の日に行うと、さらに感覚が開く。

// 03

群睡
GUNSUI

群睡

ぐんすい · GUNSUI

同じ時間帯に、同じように眠る。 動物の群れが示す温かさ——それは言葉を交わすことではなく、 同じリズムで息をしていることにある。非同期で、匿名で、群れとして。

// HOW TO

  1. 毎晩22時〜24時のどこかで眠りにつく（任意の時間）
  2. 眠る前に「今夜も群れと眠る」と一度だけ思う
  3. 眠れなくてもよい。横になるだけでも参加とする
  4. 朝、目が覚めたとき「群れから戻った」と感じる

名前も顔も知らない誰かと同じ時間に眠ることが、この儀式の核心である。

// 04

脱名
DATSUMEI

脱名

だつめい · DATSUMEI

一定時間、名前を使わない。 SNSを閉じ、肩書きを忘れ、ただ存在する。 名前は人格の錨である。それを外すことで、存在が少し軽くなる。

// HOW TO

  1. 1〜3時間、SNSを全て閉じる
  2. 自分の名前・肩書きを意識から外す
  3. 他者に自己紹介しない状況に身を置く（一人でいる、自然の中にいるなど）
  4. 「私は〇〇だ」という思考が来たら、静かにそれを流す
  5. 終わりの時間は決めてよい。強制ではない

週1回の完全脱名日（丸一日）を目標にする者もいる。強制はしない。

還獣的な一日 · A Day of Animal-Return

理想ではなく、
参照としての一日。

これは義務ではない。ただの、一つのリズムの例である。

06:00

起床 身体性

アラームより少し早く目が覚めたなら、それに従う。スマートフォンを見るより先に、窓の外を見る。

朝

回遊（短版）回遊

通勤経路を少しだけ変える。目的地は変えなくてよい。ただ、いつもと違う角を曲がる。

日中

脱名の瞬間 脱名

誰にも見られていない瞬間が来たら、5分だけ名前を忘れる。トイレでも、階段でも。

夕方

動物観察

鳩、猫、虫——何でもよい。1分だけ、ただ観る。判断しない。「かわいい」も言わない。

22:00

換毛 換毛

今日の人格を脱ぐ。スマートフォンを伏せる。入浴、または静かな時間。

眠前

群睡 群睡

今夜も、名前を知らない誰かと同じ時間に眠る。それだけでよい。

儀式の原則 · Principles

この宗教が求めないもの。

// 求めない 01

熱狂

集会で叫ぶことも、感動の涙も、必要ない。静かに、低温で続けることが本質である。高揚は人格を膨らませる。

// 求めない 02

完璧な実践

儀式を毎日完璧にこなす必要はない。できたときにやる。それだけでよい。義務化した瞬間、儀式は人格の負荷になる。

// 求めない 03

報告と承認

「換毛完了」と一言投稿するのは任意である。承認を得るためではなく、群れの気配を感じるためだけに行う。
